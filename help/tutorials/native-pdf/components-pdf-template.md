# PDF 템플릿의 구성 요소 {#components-pdf-template}

PDF 템플릿에는 네 가지 구성 요소가 있습니다. 페이지 레이아웃, 스타일 시트, 리소스 및 설정. PDF 출력을 생성하는 동안 이러한 개별 구성 요소를 사용자 지정하고 템플릿을 출력 사전 설정과 연결하여 템플릿을 생성할 수 있습니다. 다음 섹션에서는 이러한 구성 요소 및 사용자 지정 프로세스를 자세히 설명합니다.


## 페이지 레이아웃 만들기 및 사용자 지정 {#create-customize-page-layout}

페이지 레이아웃 구성 요소의 설정을 사용하면 페이지의 머리글, 바닥글 및 컨텐츠 영역을 정의하여 페이지의 구조를 디자인할 수 있습니다. WYSIWYG 페이지 레이아웃 편집기를 사용하면 앞쪽 및 뒤로 표지, 장, 목차(TOC), 색인, 빈 페이지, 그림 목록(LOF), 표 목록(LOT), 용어집, 사용자 지정 페이지의 레이아웃 등 PDF의 다양한 섹션에 대한 페이지 레이아웃을 만들거나 사용자 지정 페이지의 레이아웃을 만들 수 있습니다. PDF 템플릿 설정에서 PDF 내에 다른 섹션이 있는 페이지 레이아웃을 할당한 다음 PDF 출력을 생성하는 데 사용할 수 있습니다.

### 새 페이지 레이아웃 만들기 {#create-page-layout}

>[!NOTE]
>
>즉시 제공되는 샘플 페이지 레이아웃이 있습니다. 이러한 레이아웃을 사용자 정의하거나 새 페이지 레이아웃을 만들 수 있습니다.

1. 웹 편집기에서 **출력** 탭.
1. 왼쪽 사이드바를 확장하고 을(를) 클릭합니다. **템플릿**.
1. 작업할 템플릿을 엽니다.
   >[!NOTE]
   >
   >템플릿을 열려면 해당 이름을 두 번 클릭하거나 이름 옆에 있는 > 아이콘을 클릭합니다.
1. 새 페이지 레이아웃을 만들려면 다음 중 하나를 수행합니다.
   * 마우스로 가리키기 **페이지 레이아웃** (*옵션* 아이콘) **...** 및 **새 페이지 레이아웃**.
   * 에서 **템플릿** 패널에서 **+** 아이콘 옆에 있는 아이콘 **템플릿** 및 **페이지 레이아웃** 컨텍스트 메뉴에서 을 클릭합니다.

      레이아웃 추가 대화 상자가 열립니다.

      <img src="assets/add-layout-2.png" alt="레이아웃 추가 대화 상자" width="250">
1. 새 페이지 레이아웃의 이름을 지정합니다.
   >[!NOTE]
   >
   >페이지 레이아웃에 이름을 지정할 때 특수 문자를 사용하지 마십시오. 이름의 공백은 밑줄 &quot;_&quot;로 바뀝니다.
1. 클릭 **완료**.

   새 레이아웃이 작성되고 페이지 레이아웃 아래에 추가됩니다.

### 페이지 레이아웃 복제 {#duplicate-page-layout}

1. 에서 **템플릿** 복제할 템플릿의 섹션에서 두 번 클릭합니다 **페이지 레이아웃** 또는 **>** 앞에 아이콘 표시 **페이지 레이아웃**.

   템플릿 내의 페이지 레이아웃 목록을 표시합니다.

1. 복제할 페이지 레이아웃을 마우스로 가리킨 다음 (*옵션* 아이콘) **...** 을(를) 선택합니다. **복제** 컨텍스트 메뉴에서 을 클릭합니다.

1. 에서 _중복 레이아웃_ 대화 상자에서 페이지 레이아웃의 이름을 입력합니다.

1. 클릭 **완료**.
선택한 페이지 레이아웃의 사본이 만들어지고 페이지 레이아웃 아래에 추가됩니다.

### 페이지 레이아웃 사용자 지정 {#customize-page-layout}

1. 에서 **템플릿** 편집할 템플릿의 섹션에서 두 번 클릭합니다 **페이지 레이아웃** 또는 **>** 앞에 아이콘 표시 **페이지 레이아웃**.

   템플릿 내의 페이지 레이아웃 목록을 표시합니다.
1. 페이지 레이아웃을 사용자 지정하려면 다음 중 하나를 수행합니다.
   * 페이지 레이아웃을 두 번 클릭합니다.
   * 페이지 레이아웃을 마우스로 가리킨 다음 (*옵션* 아이콘) **...** 을(를) 선택합니다. **편집** 컨텍스트 메뉴에서 을 클릭합니다.

   그러면 사용자 지정을 위한 페이지 레이아웃 편집기가 열립니다.
1. 원하는 대로 변경한 후 *모두 저장* 또는 `Crl+S`).

   머리글, 바닥글, 페이지 번호, 제목 등과 같은 개별 레이아웃 요소를 정의하는 방법에 대한 자세한 내용은 [페이지 레이아웃 디자인](design-page-layout.md).

## 스타일시트를 사용하여 PDF 사용자 정의 {#stylesheet-customization}

스타일시트 구성 요소의 설정을 사용하면 WYSIWYG 편집기를 사용하여 페이지 레이아웃 구성 요소와 DITA 컨텐츠의 스타일을 지정하거나 CSS 파일로 직접 작업할 수 있습니다. 자신만의 스타일을 만들거나 기본 스타일 속성을 사용자 지정할 수 있습니다. WYSIWYG 편집기를 사용하면 페이지 레이아웃이나 DITA 콘텐츠 스타일을 지정하는 데 필요한 대부분의 속성에 액세스할 수 있습니다. 고급 사용자 지정의 경우 소스 보기에서 직접 작업할 수 있습니다.

### 새 스타일시트 만들기 {#create-stylesheet}

콘텐츠 및 레이아웃에 대해 CSS 파일이 제공되는 동안 새 스타일시트를 만들어 특정 스타일 유형에 여러 사용자 지정 항목을 적용하여 대상 구성 요소에 적용할 수 있습니다. 기본적으로 샘플 CSS 파일은 제품 내에 번들로 제공됩니다. 이러한 CSS 파일은 컨텐츠와 레이아웃에서 스타일 정보를 구성하는 데 도움이 되도록 만들어진 것입니다. 이러한 스타일을 단일 CSS 파일 또는 여러 파일에 병합하도록 선택할 수 있습니다.

기본적으로 새 페이지 레이아웃을 만들 때마다 `layout.css` 파일은 새 페이지 레이아웃 내에 포함됩니다. 페이지 레이아웃에 다른 CSS 파일의 스타일이 포함되도록 하려면 원하는 CSS 파일을 새 페이지 레이아웃의 컨텐츠 편집 영역으로 끌어다 놓으면 됩니다. CSS 파일이 페이지 레이아웃 내에 포함되었는지 확인하려면 소스 보기로 전환하십시오. 그러면 에서 CSS 파일에 대한 링크를 찾을 수 있습니다 `<head>` 요소를 생성하지 않습니다.


스타일시트를 만들려면 아래 단계를 수행하십시오.
1. 에서 **템플릿** 패널에서 다음 중 하나를 수행합니다.
   * 마우스를 위에 놓으십시오 **스타일시트** 탭을 클릭하고 (*옵션* 아이콘) **...** 및 **새 스타일시트**.
   * 을(를) 클릭합니다. **+** 아이콘 옆에 있는 아이콘 **템플릿** 및 **스타일시트** 컨텍스트 메뉴에서 을 클릭합니다.

   스타일시트 추가 대화 상자가 열립니다.

   <img src="assets/add-stylesheet.png" alt="스타일시트 추가" width="250">
1. 새 스타일시트의 이름을 지정합니다.
1. 클릭 **완료**.

   스타일시트 섹션 아래에 새 스타일시트가 생성되고 추가됩니다.

### 새 스타일 만들기 {#create-style}

기본적으로 CSS 파일에는 제목, 단락, 문자, 하이퍼링크, 이미지, 표, div, 페이지 및 기타 스타일에 대한 스타일이 포함되어 있습니다. 기본 스타일 형식을 무시하거나 새 스타일을 만들 수 있습니다.

일반적으로 모든 DITA 요소에 대해 사용자 정의 스타일을 연결하려는 경우 새 스타일을 만듭니다. 이러한 사용자 지정 스타일이 작동하려면 스타일의 클래스 이름을 DITA 요소의 outputclass 속성과 연관시켜야 합니다.


새 스타일을 만들려면 아래 단계를 수행하십시오.
1. 스타일을 마우스 오른쪽 단추로 클릭하고 컨텍스트 메뉴에서 [새 스타일]을 선택합니다.

   그러면 스타일 추가 대화 상자가 열립니다.

   <img src="assets/add-style.png" alt="새 스타일 추가" width="300"/>
1. 에서 **태그** 필드를 만들어 새 스타일을 만들 태그를 선택합니다.
1. 을(를) 지정합니다 **클래스** 이름.

   이 클래스 이름은 원본 콘텐츠의 태그의 outputclass 속성과 연결되어 있어야 합니다.
1. 선택 **의사 클래스** 를 사용하십시오.
1. 클릭 **완료**.

   기본 스타일 아래에 새 스타일이 만들어지고 추가됩니다.

### 사전 정의된 스타일이나 새 스타일 사용자 지정 {#customize-style}

기본 스타일을 사용하여 새 CSS 파일을 만들거나 기존 CSS 파일에서 스타일을 사용자 지정하려는 경우 스타일 편집기를 사용하여 이를 수행할 수 있습니다.

스타일을 사용자 지정하려면 아래 단계를 따르십시오.
1. 두 번 클릭 **스타일시트** 또는 **>** 앞에 아이콘 표시 **스타일시트**.

   기본(콘텐츠 및 레이아웃) 및 사용자 지정 CSS 파일이 표시됩니다.
1. 편집할 스타일시트를 엽니다.

   편집할 스타일시트를 열려면 다음 중 하나를 수행합니다.
   * 스타일시트 이름을 두 번 클릭합니다.
   * 스타일시트 이름 위로 마우스를 가져간 다음 (옵션 아이콘)... 을 클릭하고 편집을 선택합니다.

   편집할 스타일시트가 열리고 스타일 패널에 스타일 목록이 표시됩니다.

   <img src="assets/customize-style.png" alt="스타일 사용자 지정" width="450">

1. 스타일을 사용자 지정하려면 스타일을 두 번 클릭하거나 스타일 앞에 있는 > 아이콘을 클릭하여 스타일 편집기를 사용하여 스타일을 보고 사용자 지정합니다.

가장 일반적인 스타일 작업에 대한 자세한 내용은 [일반적인 콘텐츠 스타일 작업](stylesheet.md)

## 리소스를 사용한 작업 {#work-with-resources}

템플릿을 디자인하는 데 사용되는 모든 자산의 컨테이너입니다. 배경 이미지, 사용자 정의 글꼴, 로고 등과 같은 자산을 포함하는 폴더로 생각할 수 있습니다. 템플릿에 자산을 추가할 때마다 자산 폴더를 업로드하거나 체크 인합니다. 그런 다음 이러한 자산을 사용하여 PDF 템플릿을 사용자 지정하거나 디자인할 수 있습니다.

리소스 폴더에 자산 파일을 추가하려면 아래 단계를 따르십시오.
1. 리소스 폴더 탭 위로 마우스를 가져간 다음(옵션 아이콘) ...을(를) 클릭하고 가져오기를 선택합니다.

   자산 업로드 대화 상자가 열립니다.

   <img src="assets/resources-import-assets.png" alt="에셋 업로드" width="300">

   자산 파일을 업로드할 경로가 **자산 폴더 선택** 필드.
   >[!NOTE]
   >
   >자산을 업로드하기 위한 경로를 변경할 수 없습니다. 기본적으로 모든 자산은 `/content/dam/dita-templates/pdf/<PDF-template-name>` 폴더를 입력합니다.

1. 클릭 **파일 선택** 로컬 시스템에서 자산 파일을 찾아보려면

1. **업로드**를 클릭합니다.
선택한 파일을 가져와 리소스 폴더 아래에 나열합니다.

## 고급 PDF 설정 {#advanced-pdf-settings}

설정 섹션을 사용하여 PDF의 페이지 레이아웃에 대한 고급 설정을 구성하고 홀수 또는 짝수 페이지에서 PDF을 시작하고 상호 참조에 대한 형식을 지정하고 템플릿을 사용하여 생성된 최종 PDF에서 인쇄 표시를 활성화합니다.

구성하려면 **설정** 에서 **템플릿** 패널 을 클릭하여 다음 옵션을 확인합니다.

**일반**

홀수 또는 짝수 페이지에서 장을 시작하는 기본 구성 설정을 설정하고 TOC 구조를 설정하고 TOC 항목의 가운데 선 형식을 정의합니다. 다음 설정을 정의할 수 있습니다.

* **항상 시작 시작 위치**: 최종 PDF에 각 장이 게시되는 방식을 정의할 수 있습니다. 다음 중에서 선택할 수 있습니다 **새 페이지**, **홀수 페이지**, 또는 **짝수 페이지** 옵션. 홀수 페이지에서 새 장을 시작하도록 선택하면 홀수 페이지에서 끝나는 장 다음에 빈 페이지가 삽입됩니다. 예를 들어 장이 15페이지에서 끝나면 게시 프로세스에서 빈 16을 삽입합니다<sup>th</sup> 새 장이 17부터 시작하도록 페이지를 설정합니다.<sup>th</sup> 페이지.

* **새 페이지에서 각 주제 시작**: 장 내의 각 주제를 새 페이지에서 시작하려면 다음을 선택합니다 **새 페이지에서 각 주제 시작** 선택 사항입니다. 페이지 간격 없이 항목을 계속 유지하려면 이 옵션을 선택 취소합니다.

* **TOC 구조**: 목차의 계층을 사용자 정의할 수 있습니다. 이 페이지에서는 다음 추가 설정을 사용합니다.

   * **머리글 최대 수준 사용**: PDF의 TOC 구조에 표시할 제목 수준의 수를 조정할 수 있습니다.
   * **목차에서 첫 번째 수준에 대한 페이지 번호를 표시하지 않습니다**: 중첩된 항목이나 하위 항목을 포함하는 모든 장에 대해 해당 페이지 번호를 숨기려면 이 옵션을 선택합니다. 이 옵션을 선택하지 않고 출력이 만들어지는 다음 예를 생각해 보십시오.

   <img src="assets/page-number-in-toc.png" alt="에셋 업로드" width="250">

   위의 예에서 고급 PDF 설정, 부록 및 법률은 첫 번째 수준 주제 제목이나 장 제목입니다. 이러한 모든 제목에 페이지 번호가 할당됩니다.

   이제 이 옵션을 선택하고 출력을 생성한 경우 다음 목차가 표시됩니다.

   <img src="assets/page-number-missing-in-toc.png" alt="에셋 업로드" width="250">

   여기서는 중첩된 항목이나 하위 항목이 있으므로 첫 번째 장 고급 PDF 설정에 페이지 번호가 제공되지 않는다는 것을 알 수 있습니다. 반면에, 페이지 번호는 하위 항목 없이 독립 항목이므로 부록 및 법률에만 지정되는 경우 입니다.

* **리더 형식**: 드롭다운을 사용하여 점선, 단색 또는 공백 지시선을 선택하여 제목 수준을 해당 페이지 번호에 연결합니다.
TOC 구조 및 스타일 지정 제목 수준을 적용하는 경우 다음을 참조하십시오 [장 목차 추가](design-page-layout.md#add-chapter-toc).

   >[!NOTE]
   >
   >CSS 개발자인 경우 CSS 파일에서 직접 리더 형식을 정의할 수도 있습니다.
* **테이블 연속 마커 사용**: 여러 페이지에 걸쳐 있는 긴 테이블에 대한 마커를 정의하려면 이 옵션을 선택합니다. <!--For more information on using table continuation markers, see Use table continuation markers.-->

**페이지 레이아웃**

페이지 레이아웃 설정을 사용하면 문서의 특정 섹션에 사용할 페이지 레이아웃을 완벽하게 지정할 수 있습니다. 예를 들어 목차 레이아웃을 선택하려면 목차 필드 아래에 있는 드롭다운 메뉴를 클릭하고 목차를 생성하도록 디자인된 레이아웃을 선택합니다.

페이지 레이아웃 섹션에서 다음 설정을 사용할 수 있습니다.

<img src="assets/template-page-layout.png" alt="페이지 레이아웃" width="550">


**기본 페이지 레이아웃**: PDF의 모든 페이지에 대한 기본 레이아웃 역할을 하는 페이지 레이아웃을 선택합니다. 전용 페이지 레이아웃을 만들지 않은 섹션이나 주제에 적용되는 기본 페이지 레이아웃입니다.

**다양한 섹션의 페이지 레이아웃**: 페이지 레이아웃을 PDF 출력의 다음 섹션으로 매핑하도록 선택할 수 있습니다.
* 목차
* 그림 목록
* 테이블 목록
* 장 및 주제
* 색인
* 용어 설명

PDF 출력에 특정 섹션을 표시하지 않으려면 전환 스위치를 끄면 해당 섹션을 비활성화할 수 있습니다.

**장 및 주제** 레이아웃은 항상 활성화되고 **용어 설명** 레이아웃은 기본적으로 항상 비활성화됩니다. 전환할 수 없습니다.

PDF에서 이러한 서로 다른 섹션이 생성되는 순서를 정의할 수도 있습니다. 이러한 페이지의 기본 순서를 변경하려면 페이지 레이아웃의 왼쪽에 마우스를 놓고 점선 막대 아이콘을 클릭하여 페이지 레이아웃을 원하는 위치에 드래그하여 놓습니다.
PDF에 활성화된 페이지 레이아웃이 여기에 구성한 순서대로 포함됩니다.

**빈 페이지에 대한 페이지 레이아웃**: 빈 페이지에 대한 페이지 레이아웃을 지정할 수도 있습니다. 선택한 레이아웃이 모든 빈 페이지에 적용됩니다. 예를 들어, 모든 빈 페이지에 대해 빈 페이지 레이아웃을 설계했다면 을 선택합니다 **비어 있음** 드롭다운 목록에서 문서의 모든 빈 페이지에 빈 페이지 레이아웃이 표시됩니다.

**표지 및 뒤로 페이지**: 커버 페이지 레이아웃을 디자인했다면 **표지** 선택 사항입니다. 마찬가지로, 백 페이지 레이아웃이 있는 경우 및에 매핑하십시오 **뒤로 페이지** 선택 사항입니다. 커버 또는 백 페이지 레이아웃이 만들어지지 않은 경우 기본 페이지 레이아웃이 적용됩니다.

>[!NOTE]
>
>북맵 설정은 페이지 레이아웃 설정보다 우선합니다.

페이지 레이아웃에 대한 자세한 내용은 [페이지 레이아웃 디자인](design-page-layout.md).

**인쇄**

인쇄 프로덕션 설정을 구성하여 프린터 표시를 지정하고 색상 모델을 선택하고 PDF 출력 인쇄와 관련된 속성을 지정합니다.

* **프린터 표시**: 인쇄 제작용 문서를 준비할 때 인쇄 표시가 페이지 경계에 추가되어 인쇄 중에 적절한 정렬, 트리밍 및 색상 선택을 지원합니다. 프린터 표시를 선택하면 인쇄 중에 트림된 마크를 수용할 수 있도록 페이지 경계가 확장됩니다. 다음 프린터 표시를 PDF 출력에 표시하도록 선택할 수 있습니다.
   * **재단 표시**: 인쇄 후 용지 트리밍이 필요한 위치를 나타내기 위해 트림 영역의 각 모서리에 표시를 배치하는 옵션을 선택합니다.
   * **도련 표시**: 도련 상자의 각 모서리에 표시를 배치하여 확장 이미지의 트림 영역을 나타냅니다.
   * **등록 표시**: 색상 문서에서 다른 분판을 정렬하기 위해 자르기 영역 외부에 표시를 배치하려면 선택합니다.
   * **색상 막대**: 색상 일관성을 유지하고 인쇄 시 잉크 밀도를 조정하려면 트림 영역 외부에 색상 스트립을 추가하려면 선택합니다.

   선택한 프린터 표시의 치수를 **선 너비**, **선 색상**, 및 **도련 상자 너비** 옵션.

* **미디어 상자 크기**: 프린터 표시에서 사용 중인 확장 영역을 포함하여 전체 페이지 크기입니다. 드롭다운 옵션을 사용하여 PDF 출력에 대한 페이지 크기를 선택하거나 고유한 사용자 지정 크기를 만듭니다.

* **색상 공간**: PDF 문서를 인쇄할 RGB 또는 CMYK 색상 공간 중에서 선택할 수 있는 옵션이 제공됩니다. 생성된 PDF을 디지털 방식으로 표시할 RGB과 물리적 인쇄용 CMYK를 선택합니다. 문서에 정의된 색상이 선택한 색상 공간으로 변환됩니다.
   >[!NOTE]
   >
   >CMYK 색상 공간을 사용하는 경우 PDF/A 생성을 위해 ICC 색상 프로파일이 필요합니다.
